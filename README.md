## Guides

  ### Setup - 
  To setup the project and to install requisite dependencies run the following commands:
  ```
  $ pip install pipenv
  $ pipenv shell
  $ pipenv install
  ```
 
  ### To run the server -
  Place the dataset in a folder named `corpus` inside the root directory of the Project and execute:
  ```
  $ python app.py
  ```
